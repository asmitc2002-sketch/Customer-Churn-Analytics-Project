# Customer Churn Analytics & Retention Prediction

## Project Overview
This project focuses on analyzing customer churn behavior in a telecom company using data analytics, machine learning, and interactive dashboarding techniques. The objective of the project is to identify the key factors influencing customer churn and provide business recommendations to improve customer retention.

---

## Business Objective
Customer churn is one of the major challenges faced by telecom companies. This project aims to:

- Identify customers likely to churn
- Analyze churn patterns and customer behavior
- Build predictive machine learning models
- Develop an interactive Power BI dashboard for business insights

---

## Dataset Information
Dataset Used: Telco Customer Churn Dataset

The dataset contains customer information such as:
- Demographics
- Contract type
- Internet services
- Payment methods
- Monthly charges
- Customer tenure
- Churn status

## Project Workflow

### 1. Data Cleaning & Preprocessing
- Handled missing values
- Converted data types
- Removed unnecessary columns
- Encoded categorical variables

### 2. Exploratory Data Analysis (EDA)
Performed analysis on:
- Customer churn distribution
- Contract type vs churn
- Tenure vs churn
- Monthly charges vs churn
- Internet service vs churn
- Payment method vs churn
- Correlation heatmap

### 3. Machine Learning Models
Implemented:
- Logistic Regression
- Random Forest Classifier

### 4. Model Evaluation
Evaluated models using:
- Accuracy Score
- Confusion Matrix
- Classification Report

---

## Key Business Insights
- Month-to-month customers exhibited the highest churn behavior
- Customers with shorter tenure were more likely to churn
- Fiber optic users demonstrated higher churn rates
- Electronic check users showed elevated churn behavior
- Long-term contracts improved customer retention

---

## Power BI Dashboard Features
The interactive dashboard includes:
- Customer churn KPIs
- Churn distribution analysis
- Contract type analysis
- Internet service analysis
- Payment method analysis
- Customer tenure analysis
- Interactive slicers and filters

---

## Business Recommendations
- Introduce targeted retention campaigns for new customers
- Encourage customers to shift toward long-term contracts
- Improve customer experience for fiber optic users
- Promote automated payment methods to improve retention
- Develop personalized retention strategies for high-risk customers

---

## Conclusion
This project demonstrates how data analytics and machine learning can help organizations identify customer churn patterns and improve customer retention strategies through data-driven decision-making.
